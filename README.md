# HR Workforce Attrition Dashboard

## Project Overview
This data analytics project explores employee attrition trends using an Excel dashboard. The aim is to identify key factors contributing to workforce attrition and provide recommendations to improve employee retention.

## Dashboard Summary
The dashboard shows that 16% of employees left the workforce, while 84% were retained.

Key insights include:
- Employees aged 21–40 account for over 70% of attrition.
- Research & Development has the highest share of employee exits.
- Laboratory Technicians, Research Scientists, and Sales Executives show high attrition.
- Employees working overtime are more likely to leave.
- Lower monthly income appears to be linked with higher attrition in some roles.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Cleaning
- Data Analysis

## Key Recommendations
- Reduce excessive overtime to improve work-life balance.
- Focus retention strategies on employees aged 21–40.
- Improve employee engagement in Research & Development and Sales.
- Invest in training, career development, and competitive pay.
